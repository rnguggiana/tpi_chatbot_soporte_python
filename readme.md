# TPI Organización Empresarial — Chatbot de Soporte Técnico Nivel 1

## Descripción

Este proyecto corresponde al Trabajo Práctico Integrador de la materia Organización Empresarial. El objetivo es simular un proceso de mesa de ayuda interna mediante un chatbot de soporte técnico nivel 1.

El sistema permite validar usuarios, clasificar problemas frecuentes, mostrar soluciones automáticas y derivar a soporte los casos que no pueden resolverse en el primer nivel de atención.

## Proceso seleccionado

El proceso elegido es la gestión de solicitudes internas de soporte técnico. En el proceso As-Is, los empleados realizan pedidos por canales informales como WhatsApp, correo electrónico, llamadas o comentarios verbales. Esto genera falta de trazabilidad, duplicación de reclamos y dificultad para conocer el estado de cada caso.

El proceso To-Be propone incorporar un chatbot como primer filtro de atención.
